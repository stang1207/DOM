DOM Review

    DOM Cheatsheet

    • Selecting elements in the DOM

      • querySelector - it returns the first element that matches the specified selector (id, class, element).

      • querySelectorAll - it returns all the elements that match the specified selector.

      • getElementsById - it returns the element with matching id.

      • getElementsByClassName - it returns all the elements with matching class name.

    • Creating elements in the DOM

      • document.createElement(tagName) - it creates the HTML element specified by tagName

    • Adding elements and removing elements

      • Element.appendChild(nodeElement) - it appends an element as the last child of a specified element.

      • Element.removeChild(nodeElement) - it removes a child element from the DOM.

    • Modifying an element in the DOM

      • Element.setAttribute(name, value) - it sets an attribute on a specified element (class, src, href, alt, etc).

      • Element.textContent - it could be used to modify the text content of an element.

      • Element.classList.add(className) - it adds a class to an element.

      • Element.classList.remove(className) - it removes a class to an element.

      • Element.classList.toggle(className) - it toggle a class of an element (add a class if the element doesn't have that class or remove a class if the element already has that class).

      • Element.addEventListener(event, callbackFunction) - it attaches an event handler to an element.

      • Element.innerHTML - it gets or sets the HTML markup contained within an element. The use of innerHTML creates a potential security risk for your website. Malicious users can use cross-site scripting (XSS) to insert malicious code into the DOM.
